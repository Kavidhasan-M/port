Redesigning Dunzo

I participated in a contest where we were asked to improve user experience in a food ordering application. Dunzo is not focused only on food delivery but I found some UI components that brought bad experience in some situations to the users.


Things redesigned

#1 Horizontal scroll for food items

A restaurant had list of 32 food items and this food items were listed using horizontal scroll(tabs) where an user had to swipe a lot to reach the items at the last. The search box may remove the problem with horizontal scrolling, but that's when the user had already decided items to order. I redesigned with accordion which makes easy for the users to check all items available in a restaurant.

Changes

Horizontal scroll(Tabs) to vertical scroll(accordion)
